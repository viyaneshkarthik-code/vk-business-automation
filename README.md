# Local Manufacturing Shop Automation Platform

This is a Python-based data analytics platform designed to modernize operations, track daily machine production targets, and automate inventory logs for our family-owned lathe and power press industry.

## Features Built So Far
* **Daily Output Tracking:** An automated Python script to compare daily production numbers against manufacturing targets.
* **Data Grid Integration:** Uses the Python Pandas data manager to display factory machine logs cleanly in a structured table.

## Tech Stack Used
* **Language:** Python 3
* **Libraries:** Pandas
* **Editor:** Visual Studio Code
